# Payeer

A Mobile Financial Service created using MERN Stack

### Functionalities:

- Users can Send Money
- Users can Cash In + Cash Out
- Users can view their transaction history
- Users can Apply to be an Agent
- Admin Controls the System

### Technologies Used:

- MERN Stack
- Tailwind CSS
- DaisyUI
- React Helmet
- React Toastify
- Sweetalert2
